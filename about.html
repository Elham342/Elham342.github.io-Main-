About This Project

This weather application merged with a To-Do application which was built using HTML, CSS, and JavaScript. The weather app uses the OpenWeatherMap API to display real-time weather data. While the to do list app save tasks inside localStorage so they stay after refreshing the page.

Main features of the weather app include city search, saving favorite cities using local storage, and 5-day weather forecasts.

On other hand, the main features of the to-do app include adding tasks, marking as done or undo, deleting tasks, clearing all tasks,saving tasks using localStorage
